# LVRN Performance Evaluation 2026

Interactive performance evaluation form for LVRN team members.

**Open:** [lvrn-perf-eval.github.io](https://lvrn-perf-eval.github.io/)

Features:
- Self + Manager independent evaluations
- 7 competency dimensions
- Auto-save to browser
- PDF download + CSV export
- No data sent to servers (all local)
